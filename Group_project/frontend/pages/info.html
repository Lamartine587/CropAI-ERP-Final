Technical Stack

Frontend Interface

A responsive, mobile-first web portal designed for farmers using entry-level smartphones in the field.

  • JWT Authentication Guard
  • REST API Integration

Backend Servers

A robust hybrid database architecture ensuring secure user data and blazing-fast IoT telemetry logging.

  • Node.js & Express API
  • PostgreSQL (User Profiles & Auth)
  • MongoDB (Sensor Logs & AI History)

Local Edge AI

We pivoted to a zero-cost, offline-capable model to ensure farmers don't need expensive data bundles to scan crops.

  • TensorFlow.js Node integration
  • Offline matrix processing
  • High-confidence disease classification

IoT Telemetry

Affordable field nodes that constantly monitor the soil and micro-climate, alerting the dashboard when conditions drop.

  • ESP32 Microcontrollers
  • Soil Moisture & DHT11 Sensors
  • 5-second polling intervals
